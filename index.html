O que é o modelo RGB?

O modelo RGB (Red, Green, Blue) O modelo RGB é um modelo de cor usado em dispositivos eletrônicos, como monitores, câmeras digitais e televisores, para criar imagens a partir da combinação de três cores básicas: vermelho (Red), verde (Green) e azul (Blue).

Como funciona o pixel no RGB?

Os pixels RGB são os menores elementos de uma tela digital que formam as imagens que vemos em dispositivos como monitores de computador, TVs, smartphones e câmeras digitais. Cada pixel é composto por três subpixels de luz que emitem cores vermelho (R), verde (G) e azul (B), as três cores básicas do modelo RGB.

O que é o modelo CMYK?

O modelo CMYK (O modelo CMYK é outro modelo de cor usado principalmente em impressão. Ele se baseia em cores subtrativas, diferente do modelo RGB (que é aditivo). No modelo CMYK, as cores são criadas pela subtração de luz, o que significa que o modelo é projetado para sistemas que usam tintas ou pigmentos, como impressoras.

Comparação entre RGB e CMYK

O modelo RGB O modelo RGB (Red, Green, Blue) é aditivo, ou seja, cria cores adicionando luz. Ele é usado em dispositivos que emitem luz, como monitores de computador, TVs e celulares, e a mistura das cores primárias (vermelho, verde e azul) em intensidades diferentes resulta em várias cores. Quando todas as cores são combinadas no máximo, o resultado é branco. Já o modelo CMYK (Cyan, Magenta, Yellow, Key/Black) é subtrativo, ou seja, cria cores removendo luz através de tintas ou pigmentos. Ele é utilizado em processos de impressão, como em impressoras, onde as cores primárias (ciano, magenta e amarelo) são misturadas, e o preto é adicionado para profundidade. A combinação das três cores primárias no modelo CMYK tende a gerar um tom marrom ou escuro, e não branco. A principal diferença entre os dois é que RGB trabalha com luz, enquanto CMYK trabalha com pigmentos, e os dois são otimizados para tipos diferentes de dispositivos: RGB para telas e CMYK para impressão.
